# Changelog — CodeVista

All notable changes are documented here. Format follows [Keep a Changelog](https://keepachangelog.com).

## [1.2.0] — 2026-06-05

### Added
- .editorconfig and .gitattributes for consistent formatting
- SUPPORT.md, CHANGELOG.md, ROADMAP.md documentation
- PR template, dependabot config (pip + GitHub Actions)
- Stale issue/PR auto-management workflow
- Auto-labeling for PRs (codevista, tests, docker, docs, ci, config)
- New contributor welcome workflow
- Connect section with social links in README

## [1.1.0] — 2026-05-11

### Added
- Enhanced code analysis features
- Docker support with docker-compose

## [0.6.0] — 2026-03-26

## [0.5.0] — 2026-03-26

## [0.4.0] — 2026-03-26
